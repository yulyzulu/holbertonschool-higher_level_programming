====================
1. Instance Method: that prints list, but sorted(ascending sort)
====================

Import module:

    >>> print_sorted = __import__('1-my_list').print_sorted

========================

Verify list
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

>>> my_list = MyList()
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.print_sorted()
[2, 3, 5]
