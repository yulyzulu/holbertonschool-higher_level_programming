====================
3. Print square:
====================

Import module:
	>>> print_square = __import__('4-print_square').print_square

Function:
=======.

=========

    >>> print_square(5)
    #####

    #####
    #####

    >>> print_square(0.5)
    Traceback (most recent call last):
    TypeError: size must be an integer

'size' is a string:
    >>> print_square("Mary")
